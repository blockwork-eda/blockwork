The `tools` command tabulates all of the available tools from all known vendors,
listing each version and identifying which is the default.

It has no options or sub-commands.

## Usage Example

```bash
$> bw tools
┏━━━━━━━━┳━━━━━━┳━━━━━━━━━┳━━━━━━━━━┓
┃ Vendor ┃ Tool ┃ Version ┃ Default ┃
┡━━━━━━━━╇━━━━━━╇━━━━━━━━━╇━━━━━━━━━┩
│ N/A    │ gcc  │ 13.1.0  │ ✔       │
│ N/A    │ make │ 4.4     │ ✔       │
│        │      │ 4.3     │         │
└────────┴──────┴─────────┴─────────┘
```
